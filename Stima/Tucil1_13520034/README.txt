1. Deskripsi SIngkat
Program ini adalah program untuk mencari kata-kata yang tersembunyi di word puzzle

2. Requirement
    gcc compiler (dapat di cek dengan megetikkan: <gcc --version> di command line)

3. Cara menggunakan program
I. Masukkan world search puzzle ke folder test.txt
    Format world search puzzle adalah sebagai berikut:
        1. Case Sensitive sehingga disarankan untuk menggunakan full huruf kapital
        2. ukuran puzzle bebas, namun harus konsisten untuk setiap baris dan kalomnya
        3. Tedapat satu buah line kosing diantara word puzzle dengan kata yang harus dicari
        4. Tidak boleh ada spasi diantara kata yang dicari
        5. kata ke-x dengan kata ke-x+1 dipisahkan oleh sebuah enter (tidak boleh ada spasi diantara mereka)
        6. tidak boleh ada enter pada akhir file (akhir file harus berupa kata yang dicari)
II. buka program menggunakan cmd/powershell (pastikan user sudah berada di path: ______/Tubes1_13520034)
III. jika menggunakan cmd, maka ketikan run.batatau run2.bat untuk menjalankan program
     jika menggunakan powershell, ketikan ./run atau ./run2 untuk menjalankan program  
    *catatan:   run digunakan untuk menjalankan program brute force saja
                run2 digunakan untuk menjalakan program brute force dengan heuristik
